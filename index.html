CSS-only Modal

Close

This modal is opened via an anchor to #modal and styled with the :target selector. It has improved spacing and clear actions. Because this is CSS-only, the Escape key won't automatically close it — click Close or the overlay.

Tip: for full keyboard trapping and Escape-to-close behavior you'd add a tiny script. This demo stays script-free on purpose.
